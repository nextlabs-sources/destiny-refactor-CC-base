Layout:
There are 4 Tabs: parsed, warning, error, properties

parsed: 
    successful parsed entries

warning: 
    entries that you should look and fix it. However it won't block the enrollment. 
    The issues should be fixed by modifying the .def file

error: 
    very bad entries. They will block the enrollment and they have to be fixed.
    
properties:
    the enrollment properties. Useful to see what the parameters are.
