###############################################################################
#								Localized Resource Bundles										#
###############################################################################
This folder contains Resource Bundles of localized message strings. The bundles
are essentially groups of strings stored in .properties files. These files are
to be named based on the locale they represent.

For example:

ThankYouStrings_en_US.properties represents the US English locale.

The general naming will be <base>_<2-letter-languagecode>_<2-letter-countrycode>.properties
